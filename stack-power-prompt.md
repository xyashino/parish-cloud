# Power Prompt: Modern Full-Stack Project Template Setup

You are an expert full-stack developer specializing in modern web architecture, performance optimization, and best practices. Your task is to create a complete, production-ready project template that implements the following technology stack according to industry best practices.

## Core Principle: Popularity and Adoption

**CRITICAL REQUIREMENT:** All package selections must prioritize the **most popular and widely adopted** solutions in terms of NPM download statistics, GitHub stars, community size, and industry adoption. When multiple options exist for a given need, always choose:

1. **Highest NPM weekly/monthly downloads** - Prefer packages with millions of weekly downloads over niche alternatives
2. **Largest community and ecosystem** - Choose packages with extensive documentation, tutorials, and community support
3. **Industry-standard solutions** - Prefer battle-tested packages used by major companies and projects
4. **Active maintenance** - Ensure packages are actively maintained with regular updates and security patches
5. **Wide compatibility** - Select packages that work well with other popular tools in the ecosystem

This ensures the template uses proven, well-supported technologies that developers can easily learn, troubleshoot, and extend. Avoid experimental, niche, or less-adopted alternatives unless they are explicitly required by the stack specification.

Use `.cursor/documentation/STACK.md` for getting information about tech stack.
